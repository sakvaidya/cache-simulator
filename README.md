# Cache Simulator

A set-associative cache simulator with LRU, FIFO, and RAND replacement policies.
Built with Python and Tkinter.

## Team
- Person A: Core cache structure, RAND policy, config UI
- Person B: LRU policy, cache contents visualization
- Person C: FIFO policy, memory input UI, statistics panel

## How to Run
```
python main.py
```

## Features
- Set-associative cache simulation
- Replacement policies: RAND, LRU, FIFO
- Step-by-step or run-to-completion simulation
- Live cache contents and statistics display
